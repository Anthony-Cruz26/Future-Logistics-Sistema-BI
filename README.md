рҹ“Ӣ DescripciГіn del Proyecto

Sistema automatizado de **Inteligencia de Negocios** para **FutureLogistic S.A.**, empresa de logГӯstica y distribuciГіn de azГәcar en Ecuador.

El sistema simula, extrae, transforma y carga datos desde un sistema SAT (Sistema de Almacenamiento y Transporte) hacia un **Data Warehouse en SQL Server** con modelo estrella, para posteriormente visualizar KPIs logГӯsticos en **Power BI**.

рҹҺҜ Objetivos del Proyecto

1. **Gestionar** los indicadores de desempeГұo (KPIs) cruciales para el proceso operativo logГӯstico
2. **Construir** la arquitectura tГ©cnica del sistema ETL-BI
3. **Automatizar** la extracciГіn programada de datos desde SAP hacia SQL Server
4. **DiseГұar** dashboards interactivos en Power BI
